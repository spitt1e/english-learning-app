# English Learning App 📚

Интерактивное веб-приложение для изучения английского языка с объяснением грамматических правил, практическими упражнениями и системой пополнения словарного запаса.

## Открыть с телефона

Приложение уже опубликовано в интернете — компьютер не нужен:

**https://spitt1e.github.io/english-learning-app/**

### Как пользоваться на телефоне

1. Откройте ссылку выше в Safari (iPhone) или Chrome (Android).
2. Добавьте на домашний экран, чтобы открывать как приложение:
   - **iPhone (Safari):** «Поделиться» → «На экран „Домой“».
   - **Android (Chrome):** меню ⋮ → «Установить приложение» или «Добавить на главный экран».
3. Все слова, прогресс и чеклист сессии сохраняются в браузере телефона.

Данные хранятся отдельно на каждом устройстве. Если очистите данные сайта в настройках браузера, прогресс на этом телефоне будет удалён.

## Возможности

### 📖 Грамматика
- 6 основных грамматических тем с подробными объяснениями
- Present Simple, Present Continuous, Past Simple
- Артикли (a, an, the)
- Conditionals (условные предложения)
- Modal Verbs (модальные глаголы)
- Примеры использования с подсветкой

### ✍️ Практика
- Три типа упражнений:
  - Заполнить пропуски
  - Выбрать правильный ответ
  - Перевод
- Мгновенная обратная связь
- Подсчёт правильных ответов

### 📝 Словарь
- Добавление собственных слов с переводами
- Возможность добавлять примеры предложений
- Система флеш-карточек для запоминания
- Отслеживание выученных слов

### 📊 Прогресс
- Статистика выученных слов
- Количество выполненных упражнений
- Процент правильных ответов
- График активности за неделю
- Помодоро-таймер и чеклист сессии

## Как использовать на компьютере

1. Откройте ту же ссылку в браузере **или** файл `index.html` локально.
2. Изучайте грамматику, практикуйтесь, пополняйте словарь и следите за прогрессом.

## Технические детали

- **Чистый HTML/CSS/JavaScript** — не требует установки зависимостей
- **LocalStorage** — данные сохраняются в браузере
- **PWA** — можно установить на телефон и пользоваться офлайн
- **Адаптивный дизайн** — удобное меню на мобильных устройствах
- **GitHub Pages** — доступ по ссылке в любое время

## Структура файлов

english-learning-app/

## Данные

Все данные (словарь и прогресс) сохраняются в LocalStorage браузера. Если вы очистите кэш браузера, данные будут удалены.

## Расширение приложения

Приложение легко расширить:
- Добавить новые грамматические правила в объект `grammarRules` в `app.js`
- Добавить новые упражнения в объект `const_exercises`
- Изменить стили в `styles.css`
