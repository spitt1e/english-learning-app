# English Learning App 📚

Интерактивное веб-приложение для изучения английского языка с объяснением грамматических правил, практическими упражнениями и системой пополнения словарного запаса.

## Возможности

### 📖 Грамматика
- 6 основных грамматических тем с подробными объяснениями
- Present Simple, Present Continuous, Past Simple
- Артикли (a, an, the)
- Conditionals (условные предложения)
- Modal Verbs (модальные глаголы)
- Примеры использования с подсветкой

### ✍️ Практика
- Три типа упражнений:
  - Заполнить пропуски
  - Выбрать правильный ответ
  - Перевод
- Мгновенная обратная связь
- Подсчёт правильных ответов

### 📝 Словарь
- Добавление собственных слов с переводами
- Возможность добавлять примеры предложений
- Система флеш-карточек для запоминания
- Отслеживание выученных слов

### 📊 Прогресс
- Статистика выученных слов
- Количество выполненных упражнений
- Процент правильных ответов
- График активности за неделю

## Как использовать

1. **Откройте файл `index.html` в браузере**
   - Просто дважды кликните на файл или откройте его через браузер

2. **Изучайте грамматику**
   - Перейдите в раздел "Грамматика"
   - Выберите интересующую тему
   - Изучите правила и примеры

3. **Практикуйтесь**
   - Перейдите в раздел "Практика"
   - Выберите тип упражнения
   - Отвечайте на вопросы и получайте мгновенную обратную связь

4. **Пополните словарь**
   - Перейдите в раздел "Словарь"
   - Нажмите "+ Добавить слово"
   - Введите английское слово, русский перевод и пример
   - Используйте флеш-карточки для запоминания

5. **Следите за прогрессом**
   - Раздел "Прогресс" показывает вашу статистику
   - График активности отображает занятия за неделю

## Технические детали

- **Чистый HTML/CSS/JavaScript** - не требует установки зависимостей
- **LocalStorage** - все данные сохраняются в браузере
- **Адаптивный дизайн** - работает на мобильных устройствах
- **Без интернета** - приложение работает офлайн

## Данные

Все данные (словарь и прогресс) сохраняются в LocalStorage браузера. Если вы очистите кэш браузера, данные будут удалены.

## Расширение приложения

Приложение легко расширить:
- Добавить новые грамматические правила в объект `grammarRules` в `app.js`
- Добавить новые упражнения в объект `const_exercises`
- Изменить стили в `styles.css`
